# PoliAgenda Pro
PWA offline-first para biblioteca normativa, agenda académica/profesional, analizador de comunicados, pendientes, portafolio protegido con PIN y respaldos.

## Instalación
Subir esta carpeta a GitHub Pages o Vercel. Abrir el enlace en Chrome/Brave y usar “Instalar app”.

## Pruebas obligatorias antes de cargar datos reales
1. Crear pendientes ficticios, cerrar y abrir.
2. Crear agenda ficticia, cerrar y abrir.
3. Crear PIN y dato ficticio en portafolio.
4. Exportar respaldo JSON.
5. Borrar datos de prueba e importar respaldo.
6. Probar sin internet.
